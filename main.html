--変数について--

変数とは

まずは変数について理解していきましょう。変数とはデータを一時的にしまっておくための箱です。 データを箱の中に収納しそれを引き出しのようにして使うのです。また変数(箱)を作るには「宣言」というものが必要です。 例として下のコードを見てください。

let name = "Alice"; console.log(name); //nameの内容を表示する。

このコードでは"name"という変数に"Alice"というデータをいれました。簡単にいいかえるならnameという引き出しにAliceという物を収納したのです。 最初についている"let"が「宣言」と呼ばれるものです。JavaScriptの宣言には主に3種類があります。"var","let","const"です。それぞれ軽く解説します。

ver

JavaScriptの古いバージョンから存在する宣言。宣言する変数の使える範囲を示しています。 verの場合は関数内外関係なくプログラム全体で宣言した変数を使用できます。ただ、変数の変更が容易なため思わぬところで 変数が変更されバグの原因になってしまうことがあります。

let

verの新しいバージョンです。本質はvarとは変わりありませんが宣言内容の変更が容易でなくなりバグの起きにくさが向上しています。

const

constはverやletとは違い変数の内容変更ができない宣言となります。バグが起きにくくいですがverやletと同様の手口で値の変更を行うと値を変更しようとした部分は エラー判定になるので注意してください。

--演算子について--

演算子とは

演算子ときいてとても難しい印象を持つ人もいるかもしれませんが基礎はとても単純なものです。 演算子とはいわゆる四則演算や等号です。＋,-,×,÷,=などの記号のことです。ただし一部プログラム上で使用できない記号もあるため紹介します。

記号の変化

  • かける(×)---かける(*) 2 × 3 = 2 * 3
  • 割る(÷)----割る(/) 6 ÷ 2 = 6 / 3
  • 余り(...)----余り(%) 余りを求める場合は÷記号の代わりに%記号を入れ余りを求める。

実際に使用してみましょう。今回の計算式では答えがすべて10になるような式を使用しています。

let a = 7 + 3; let b = 13 - 3; let c = 2 * 5; let d = 50 / 5; let e = 21 % 11;

(=)を使った演算子

=を使った演算子はいくつか種類があります。細かく紹介するときりがないので今回は3つ紹介します。

  • (=) 代入するという意味です。x = 3 この場合はxに3が代入されています。
  • (==) 等しいという意味です。x == y この場合はxとyが等しいとされています。
  • (+=) 足し算をして代入という意味です。let x = 10; x += 5; この場合10に5を足してから代入される のでxの値は15ということになります。

--制御構文について--

制御構文とは

制御構文とは、プログラムの実行の流れをコントロールする構文です。特定の条件下でコードを実行したり、繰り返し 処理を行うコードが制御構文なのです。では、具体的にどのようなものなのでしょうか、今回は具体例を3つ挙げて紹介しましょう。

if else文

条件分岐といえばこれ。「もし...なら...する。そうでないなら...する。」といった御描き方をします。

if (a = 10){b = 10} else{b = 10}

if文の形はif(条件){実行内容}else{条件外だった場合の実行内容}です。またelseをelifに変えることで複数条件にすることもできます。

switch文

「この場合は...する。この場合は...する。」というようにいくつかのケースに分けて 実行内容を変えるのがswitchです。

let fruit = "リンゴ"; switc(fruit){ case"リンゴ": console.log("これはリンゴだ"); break; case"バナナ": console.log("これはバナナだ"); break; default: console.log("なんだこれは") }

このコードでは入力されているのがリンゴかバナナかはたまた別のものなのかを判別しています。 形としてはcase"ケース1":ケース1だった場合何をするか;break; default:ケース外の場合何をするのか

for文

繰り替えしを行う構文です。これは実例を見て理解するほうが早いと思います。

for(let i = 0;i < 5;i++) console.log(i);

このコードでは0から4までの数字を表示するようになっています。繰り返しの処理手順としては 「i=0」からスタートi=0なのでi < 5が成り立つたつためコードを実行しiに1を足す。 このようにしていくとi=5の時がやってきます。i < 5が成り立たなくなるため繰り返しがストップします。 最後にfor文の形を示しておきます。for(初期値;条件;増減){実行内容}

--関数について--

関数とは

関数とは何かしらのタスクを実行するためのコードの集合のことです。

function 関数名(){実行したいコード}

関数を定義するにはこのように「function」を使用します。その後ろには関数を象徴するような名前を記入してください。どのようなものでも構いませんが コードを見たときにすぐわかるような名前が好ましいです。次に{}の中に実行するコードを詰めていきます。プログラム内で関数が呼び出されたとき{}内のプログラムが すべて引き出されます。

戻り値について

関数内では計算を行うことができその計算結果を「戻り値」といいます。

function add(a,b){return a + b;}

この関数ではa,bを作り、その足し算結果がreturn(戻り値)として収納されています。しかし、a,bに 具体的な数字は入力されていないため戻り値は発生していません。では、ここに数値を入力していましょう。

let result = add(3,5);

このようにa,bに3,5と数値を入れると戻り値が発生しresultに計算結果が収納されます。このように関数を使用することでコードの整理や管理がしやすくなり、 繰り返しの処理も楽に行うことができます。

--オブジェと配列について--

配列について

配列とは、データの集まりのようなものです。配列は[]で囲まれており。決めた名前の後ろに"名前"[データの順番]で呼び出すことができます。

let deta = [a,b,c,d];

"deta"という名前の配列を作ってみました。呼び出す際はdate[番号]で呼び出せるのですが 気を付けるべきなのがこの「番号」です。配列における番号は通常と違い左から0,1,2,3という 順になります。そのためaを呼び出したい場合はdate[1]ではなくdate[0]と入力しなければなりません。 これが配列における最大の注意点です。

オブジェクトについて

オブジェクトとは関連データの集まりのことを言います。例えばAさんのプロフィールを作るとしましょう。 この際に入力するデータは年齢、性別、血液型など様々なものがあります。これらAさんにかかわるデータを ひっくるめて集めたものをオブジェクトと呼ぶのです。

let Mr.A = {name:"A",age:20,living:"Osaka"}

Aさんのプロフィールデータのオブジェクトを作ってみました。このコードの"name"や"age"は「キー」と呼ばれ 使用される際の名前です。その後ろは「値」で「キー」と「値」はMr.Aというオブジェクト内で結び付けられています。 オブジェクト内のデータを使用したい場合は(オブジェクト名.キー);のようにして呼び出すことができます。 配列同様、コードの整理などで役立ちます。
